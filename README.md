# URLScans Community Phishing Feed

- **Updated every 12 hours**
- Non-commercial use only

## About

Community phishing feed from [URLScans](https://urlscans.com). Contains URLs and domains flagged as malicious or suspicious by our scanning service and threat intelligence feeds.

## Feed Format

- One URL or domain per line (plain text)
- Direct link: `https://raw.githubusercontent.com/URLscans/public_feed/main/feed.txt`

## Usage Example

curl -s https://raw.githubusercontent.com/URLscans/public_feed/main/feed.txt | head -20
